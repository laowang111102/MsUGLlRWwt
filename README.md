# 项目简介 python764

本项目是基于Python开发的摄影平台交流系统，旨在为摄影爱好者提供一个互动交流的平台。以下是该系统的相关介绍。

## 技术栈

- 后端：Python 3.x，Flask框架
- 数据库：MySQL
- 前端：HTML，CSS，JavaScript，Vue.js

## 功能模块

1. 用户模块
   - 用户注册与登录
   - 用户信息管理
   - 用户作品发布与展示
   - 用户互动与评论

2. 作品模块
   - 作品分类
   - 作品搜索与筛选
   - 作品详情展示

3. 交流模块
   - 私信功能
   - 作品评论区
   - 话题讨论区

## 系统角色

- 普通用户：可以浏览、评论、发布作品，与其他用户互动
- 管理员：负责管理用户和作品，维护系统秩序

## 运行环境

- Python 3.x
- Flask框架
- MySQL数据库
- Vue.js

## 部署步骤

1. 安装Python 3.x及Flask框架
2. 安装并配置MySQL数据库
3. 克隆项目代码并进入项目目录
4. 运行`pip install -r requirements.txt`安装依赖
5. 运行`python manage.py runserver`启动开发服务器

## 目录结构

```
.
├── app
│   ├── __init__.py
│   ├── models.py
│   ├── views.py
│   ├── static
│   └── templates
├── config.py
├── manage.py
└── requirements.txt
```

## 核心亮点

- 使用Python+Flask框架，易于开发和维护
- 前后端分离，便于扩展和优化
- 提供丰富的摄影作品分类，满足不同用户的需求
- 强大的互动功能，提高用户活跃度
- 灵活的话题讨论区，激发用户创作灵感
- 严格的用户权限管理，保障系统安全稳定
